# mindcontrol-docker

A docker container for [mindcontrol](https://github.com/akeshavan/mindcontrol)

#### WORK IN PROGRESS

## Usage

```
docker run -it --rm -p 3000:3000 pwighton/mindcontrol
```

Then browse to [localhost:3000](http://localhost:3000)
